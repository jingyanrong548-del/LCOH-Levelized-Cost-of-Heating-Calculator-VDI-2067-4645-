这是一个专业的“平准化供热成本 (LCOH) 计算器”单页 Web 应用。

它基于德国工程师协会 VDI 2067 & 4645 标准，专门用于详细对比工业热泵（如余热回收型、环境热源型）与传统供热方案（电热锅炉、天然气锅炉、生物质锅炉）的长期经济性。

用户需输入通用参数（年热需、折现率等）、能源价格和各技术方案的设备参数（投资成本、运维费率、效率或SPF）。

该工具的核心功能是计算并输出：

LCOH 对比图：使用 Chart.js 生成按成本排序的条形图，结果一目了然。

详细数据表：分解展示年均投资、运维及运行成本。

关键指标分析：若对比天然气，会自动计算“电气价格比”和“临界点 SPF”，提供深度决策依据。

应用界面使用 Tailwind CSS 构建，风格现代、交互简洁，是一个功能完善的工程经济性分析工具。
